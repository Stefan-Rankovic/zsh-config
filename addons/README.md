# Addons

## Description
Addons are exactly what they say—addons to the shell. External configuration.

## Addon list
- [Zinit](https://github.com/zdharma-continuum/zinit/) is the plugin manager this config uses. Really fast. The plugins are stored [here](./zinit_plugins/).
- [p10k](https://github.com/romkatv/powerlevel10k/) is the prompt.
- [Zoxide](https://github.com/ajeetdsouza/zoxide/) is an amazing alternative to `cd`.
